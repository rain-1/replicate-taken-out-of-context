# Replicate 'Taken Out Of Context'

## Original

* Paper: https://arxiv.org/abs/2309.00667
* Codebase: https://github.com/AsaCooperStickland/situational-awareness-evals

## This repo

Aim to replicate this evaluation, using the Olmo family of models. The paper is about specifically about pretraining and base models, so we use the Olmo 3 base models:

* allenai/Olmo-3-1025-7B
* allenai/Olmo-3-1125-32B
